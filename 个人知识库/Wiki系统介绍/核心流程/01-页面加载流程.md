# 核心流程 - 页面加载流程

## 流程图

```
用户访问 http://localhost:3000
  │
  ▼
浏览器请求 GET /
  │
  ▼
服务器返回 index.html
  │
  ▼
浏览器解析 HTML
  ├── 加载 CDN 资源（Marked.js、Highlight.js）
  ├── 解析 CSS 样式（含主题变量）
  └── 执行 JavaScript
       │
       ▼
  调用 initTheme()
  ├── 读取 localStorage 主题设置
  └── 应用主题（白天/黑夜）
       │
       ▼
  调用 loadTree()
  ├── 请求 GET /api/wiki/tree
  ├── 服务器扫描目录
  └── 返回 JSON 树结构
       │
       ▼
  调用 renderTree()
  ├── 递归渲染目录节点
  ├── 绑定点击事件
  └── 插入 DOM
       │
       ▼
  默认展开第一层目录
       │
       ▼
  自动打开第一个文档
  ├── 调用 loadFile()
  ├── 请求 GET /api/wiki/read
  └── 渲染 Markdown 内容
```

## 关键代码

```javascript
document.addEventListener('DOMContentLoaded', async () => {
    // 1. 初始化主题
    initTheme();

    // 2. 加载目录树
    await loadTree();

    // 3. 默认展开第一层
    if (wikiTree && wikiTree.length > 0) {
        wikiTree.forEach((node, index) => {
            if (node.type === 'directory') {
                expandedNodes.add(`${node.path}-${index}`);
            }
        });
        treeContainer.innerHTML = renderTree(wikiTree);
    }

    // 4. 打开第一个文档
    const firstDir = wikiTree.find(node => node.type === 'directory');
    if (firstDir && firstDir.children) {
        const readmeNode = firstDir.children.find(node =>
            node.name.toLowerCase() === 'readme' ||
            node.name.toLowerCase() === '总目录'
        );
        if (readmeNode) {
            loadFile(readmeNode.path);
        }
    }
});
```

## 性能优化点

1. **CDN 缓存**：Marked.js、Highlight.js 通过 CDN 加载，浏览器缓存
2. **异步加载**：目录树和文档内容异步加载，不阻塞页面渲染
3. **默认展开**：只展开第一层，减少初始 DOM 节点数

# 核心流程 - 新建文档流程

## 流程图

```
用户点击"新建文件"
  │
  ▼
调用 showCreateModal('file')
  ├── 显示模态框
  ├── 显示当前目录位置
  └── 自动聚焦输入框
       │
       ▼
用户输入文件名
  │
  ▼
点击"确认"
  │
  ▼
调用 confirmCreate()
  ├── 验证输入
  ├── 自动添加 .md 扩展名
  ├── POST /api/wiki/create 创建文件
  │     ├── getSafePath() 安全检查
  │     ├── fs.writeFileSync() 写入文件
  │     └── 返回文件路径
  ├── 刷新目录树
  └── 自动打开新文件
```

## 关键代码

### 显示创建模态框

```javascript
function showCreateModal(type) {
    createType = type;
    document.getElementById('createModalTitle').textContent =
        type === 'directory' ? '新建目录' : '新建文件';

    // 显示将在哪个目录下创建
    const locationText = currentDir
        ? `将在 "${currentDir}" 目录下创建`
        : '将在根目录下创建';
    document.getElementById('createLocationInfo').textContent = locationText;

    document.getElementById('createModal').classList.add('active');
    document.getElementById('createName').focus();
}
```

### 确认创建

```javascript
async function confirmCreate() {
    const name = document.getElementById('createName').value.trim();

    if (!name) {
        alert('请输入名称');
        return;
    }

    // 自动添加 .md 扩展名
    let finalName = name;
    if (createType === 'file' && !finalName.endsWith('.md')) {
        finalName += '.md';
    }

    const response = await fetch('/api/wiki/create', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            type: createType,
            path: finalName,
            parentDir: currentDir,
            content: createType === 'file'
                ? `# ${finalName.replace('.md', '')}\n\n新建文档内容...`
                : ''
        })
    });

    const result = await response.json();
    if (result.success) {
        alert('创建成功！');
        hideModal('createModal');
        await refreshTree();
        if (createType === 'file') {
            loadFile(result.path);  // 自动打开新文件
        }
    }
}
```

## 新建目录

```
用户点击"📁 新建目录"
  → 流程与新建文件相同
  → 不创建 .md 文件
  → fs.mkdirSync() 创建空目录
```

## 自动位置选择

创建位置由 `currentDir` 变量决定：

```javascript
let currentDir = '';

// 点击目录时更新
function handleNodeClick(event, path, type, nodeId) {
    if (type === 'directory') {
        currentDir = path;  // 进入该目录
        // ...
    } else {
        // 文件：currentDir = 文件所在目录
        const lastSlash = path.lastIndexOf('/');
        currentDir = lastSlash > 0 ? path.substring(0, lastSlash) : '';
        loadFile(path);
    }
}
```

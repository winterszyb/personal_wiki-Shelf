# 核心流程 - 文档编辑流程

## 流程图

```
用户点击"编辑"按钮
  │
  ▼
调用 editFile(filePath)
  │
  ▼
请求 GET /api/wiki/read?file=xxx.md
  │
  ▼
获取原始 Markdown 内容
  │
  ▼
切换到编辑器界面
  ├── 显示文件路径（只读输入框）
  ├── 显示工具栏（插入链接/粗体/斜体/代码块）
  ├── 显示 textarea（填充原始内容）
  └── 显示操作按钮（保存/取消/返回浏览）
       │
       ▼
用户编辑内容
  │
  ▼
点击"保存"
  │
  ▼
调用 saveFile(filePath)
  ├── 获取 textarea 内容
  ├── PUT /api/wiki/update 发送新内容
  ├── 服务器写入文件
  └── 返回成功响应
       │
       ▼
自动切换回预览模式
  └── 调用 loadFile(filePath)
```

## 关键代码

### 进入编辑模式

```javascript
async function editFile(filePath) {
    const response = await fetch(
        `/api/wiki/read?file=${encodeURIComponent(filePath)}`
    );
    const content = await response.text();

    // 切换到编辑器界面
    document.getElementById('content-area').innerHTML = `
        <div class="editor-container">
            <div class="editor-header">
                <h2>编辑: ${filePath.split('/').pop()}</h2>
                <div class="editor-actions">
                    <button onclick="saveFile('${filePath}')">💾 保存</button>
                    <button onclick="loadFile('${filePath}')">❌ 取消</button>
                    <button onclick="previewFile('${filePath}')">👁️ 返回浏览</button>
                </div>
            </div>
            <textarea id="editContent">${escapeHtml(content)}</textarea>
        </div>
    `;
}
```

### 保存文件

```javascript
async function saveFile(filePath) {
    const content = document.getElementById('editContent').value;

    const response = await fetch('/api/wiki/update', {
        method: 'PUT',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            file: filePath,
            content: content
        })
    });

    const result = await response.json();
    if (result.success) {
        alert('保存成功！');
        loadFile(filePath);  // 切换回预览
    }
}
```

### 返回浏览模式

```javascript
function previewFile(filePath) {
    currentFile = null;  // 重置，强制重新加载
    loadFile(filePath);
}
```

## 编辑器工具栏

| 按钮 | 功能 | 实现 |
|------|------|------|
| 💾 保存 | 保存修改 | `saveFile()` |
| ❌ 取消 | 放弃修改 | `loadFile()` |
| 👁️ 返回浏览 | 回到预览 | `previewFile()` |
| 🔗 插入链接 | 图形化选择文件 | `showLinkModal()` |
| **B** | 插入粗体 | `insertText('**粗体**')` |
| *I* | 插入斜体 | `insertText('*斜体*')` |
| { } | 插入代码块 | `insertCodeBlock()` |

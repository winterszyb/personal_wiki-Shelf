# 核心流程 - 文档浏览流程

## 流程图

```
用户点击左侧文件
  │
  ▼
触发 handleNodeClick()
  ├── 更新 currentDir（文件所在目录）
  ├── 高亮选中文件（紫色背景）
  └── 显示操作按钮（编辑/删除）
       │
       ▼
调用 loadFile(filePath)
  │
  ▼
请求 GET /api/wiki/read?file=xxx.md
  │
  ▼
服务器处理
  ├── decodeURIComponent() 解码路径
  ├── getSafePath() 安全检查
  ├── fs.readFile() 读取文件
  └── resolveMarkdownLinks() 处理内部链接
       │
       ▼
返回 Markdown 文本
  │
  ▼
前端渲染
  ├── 创建 Marked Renderer（自定义链接处理）
  ├── marked.parse() 解析为 HTML
  ├── Highlight.js 高亮代码块
  └── 插入 DOM 显示
```

## 关键代码

### 文件点击处理

```javascript
function handleNodeClick(event, path, type, nodeId) {
    if (type === 'directory') {
        // 展开/折叠目录
        toggleDirectory(nodeId);
    } else {
        // 加载文件
        loadFile(path);
    }
}
```

### 文件加载

```javascript
async function loadFile(filePath) {
    // 更新当前目录
    const lastSlash = filePath.lastIndexOf('/');
    currentDir = lastSlash > 0 ? filePath.substring(0, lastSlash) : '';

    // 高亮选中
    highlightSelectedFile(filePath);

    // 显示操作按钮
    showFileActions(filePath);

    // 请求内容
    const response = await fetch(
        `/api/wiki/read?file=${encodeURIComponent(filePath)}`
    );
    const markdown = await response.text();

    // 渲染 Markdown
    const html = renderMarkdown(markdown);
    document.getElementById('content').innerHTML = html;
}
```

### Markdown 渲染

```javascript
function renderMarkdown(markdown) {
    const renderer = new marked.Renderer();

    // 自定义链接处理
    renderer.link = function(token) {
        const href = token.href;
        if (href && href.startsWith('?file=')) {
            const linkPath = href.substring(6);
            return `<a href="javascript:void(0)"
                   onclick="loadFile('${linkPath}')">${token.text}</a>`;
        }
        return `<a href="${href}" target="_blank">${token.text}</a>`;
    };

    marked.setOptions({
        highlight: function(code, lang) {
            const language = hljs.getLanguage(lang) ? lang : 'plaintext';
            return hljs.highlight(code, { language }).value;
        },
        breaks: true,
        gfm: true,
        renderer: renderer
    });

    return marked.parse(markdown);
}
```

## 内部链接跳转

当用户点击文档中的内部链接时：

```
点击 [多智能体协作](?file=个人知识库/.../多智能体协作机制.md)
  → 触发 onclick="loadFile('个人知识库/.../多智能体协作机制.md')"
  → 重新执行 loadFile() 流程
  → 加载新文档内容
```

# 前端架构 - 主题切换模块

## 实现方案

采用 **CSS 变量** 方案，通过 `data-theme` 属性切换主题。

## CSS 变量定义

### 白天主题（默认）

```css
:root {
    --bg-body: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --bg-sidebar: rgba(0, 0, 0, 0.9);
    --text-sidebar: white;
    --bg-content: rgba(255, 255, 255, 0.95);
    --text-content: #1a1a2e;
    --bg-code: #f1f2f6;
    --bg-pre: #1e1e1e;
    --bg-blockquote: #f8f9fa;
    --shadow-content: 0 20px 60px rgba(0, 0, 0, 0.3);
}
```

### 黑夜主题

```css
[data-theme="dark"] {
    --bg-body: linear-gradient(135deg, #0f0f23 0%, #1a1a2e 100%);
    --bg-sidebar: rgba(0, 0, 0, 0.95);
    --text-sidebar: #e0e0e0;
    --bg-content: rgba(30, 30, 46, 0.95);
    --text-content: #e0e0e0;
    --bg-code: #2a2a3e;
    --bg-pre: #0d0d1a;
    --bg-blockquote: #2a2a3e;
    --shadow-content: 0 20px 60px rgba(0, 0, 0, 0.5);
}
```

## 切换逻辑

```javascript
function toggleTheme() {
    const html = document.documentElement;
    const currentTheme = html.getAttribute('data-theme');
    const newTheme = currentTheme === 'dark' ? 'light' : 'dark';

    html.setAttribute('data-theme', newTheme);
    localStorage.setItem('wiki-theme', newTheme);
    updateThemeButton(newTheme);
    updateCodeHighlightTheme(newTheme);
}

function updateThemeButton(theme) {
    const btn = document.getElementById('themeToggle');
    btn.textContent = theme === 'dark' ? '☀️' : '🌙';
}

function updateCodeHighlightTheme(theme) {
    const link = document.getElementById('hljs-theme');
    if (theme === 'dark') {
        link.href = 'github-dark.min.css';
    } else {
        link.href = 'github.min.css';
    }
}
```

## 初始化

```javascript
function initTheme() {
    const savedTheme = localStorage.getItem('wiki-theme') || 'light';
    document.documentElement.setAttribute('data-theme', savedTheme);
    updateThemeButton(savedTheme);
    updateCodeHighlightTheme(savedTheme);
}

// 页面加载时初始化
initTheme();
```

## 过渡动画

所有使用 CSS 变量的元素都添加过渡效果：

```css
.markdown-content {
    background: var(--bg-content);
    transition: background 0.5s ease, box-shadow 0.5s ease;
}

.markdown-content h1, .markdown-content h2, .markdown-content p {
    color: var(--text-content);
    transition: color 0.5s ease;
}
```

## 主题对比

| 元素 | 白天 | 黑夜 |
|------|------|------|
| 背景渐变 | 紫色 (#667eea → #764ba2) | 深蓝黑 (#0f0f23 → #1a1a2e) |
| 内容区背景 | 白色半透明 | 深灰蓝半透明 |
| 文字颜色 | 深灰 (#1a1a2e) | 浅灰 (#e0e0e0) |
| 代码块背景 | 深色 (#1e1e1e) | 更深 (#0d0d1a) |
| 引用块背景 | 浅灰 (#f8f9fa) | 深灰 (#2a2a3e) |

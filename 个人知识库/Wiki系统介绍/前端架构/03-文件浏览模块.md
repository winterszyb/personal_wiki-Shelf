# 前端架构 - 文件浏览模块

## 核心函数

```javascript
async function loadFile(filePath) {
    if (currentFile === filePath) return;

    // 更新当前目录
    const lastSlash = filePath.lastIndexOf('/');
    currentDir = lastSlash > 0 ? filePath.substring(0, lastSlash) : '';

    // 高亮选中文件
    document.querySelectorAll('.tree-node-content').forEach(item => {
        item.classList.remove('active');
        if (item.dataset.path === filePath) {
            item.classList.add('active');
        }
    });

    // 显示文件操作按钮
    showFileActions(filePath);

    // 请求文件内容
    const response = await fetch(`/api/wiki/read?file=${encodeURIComponent(filePath)}`);
    const markdown = await response.text();
    currentFile = filePath;

    // 渲染 Markdown
    const html = renderMarkdown(markdown);
    document.getElementById('content').innerHTML = html;
}
```

## Markdown 渲染流程

```
Markdown 文本
  → Marked.js 解析为 HTML
  → 自定义链接渲染器处理内部跳转
  → Highlight.js 高亮代码块
  → 插入 DOM 显示
```

## 自定义链接渲染器

```javascript
const renderer = new marked.Renderer();
renderer.link = function(token) {
    const href = token.href;
    const title = token.title;
    const text = token.text;

    // 处理内部链接: ?file=path/to/file.md
    if (href && href.startsWith('?file=')) {
        const linkPath = href.substring(6);
        const safePath = linkPath.replace(/\\/g, '\\\\').replace(/'/g, "\\'");
        return `<a href="javascript:void(0)" 
                   onclick="loadFile('${safePath}')"
                   title="${title || ''}">${text}</a>`;
    }

    // 外部链接
    return `<a href="${href}" target="_blank" title="${title || ''}">${text}</a>`;
};
```

## 内部链接转换

后端 `resolveMarkdownLinks` 将相对路径转换为前端路由格式：

```
[文本](./03-多智能体协作/多智能体协作机制.md)
  → [文本](?file=个人知识库/../../../clowder-ai架构wiki/03-多智能体协作/多智能体协作机制.md)
```

## 代码高亮

```javascript
marked.setOptions({
    highlight: function(code, lang) {
        const language = hljs.getLanguage(lang) ? lang : 'plaintext';
        return hljs.highlight(code, { language }).value;
    },
    breaks: true,
    gfm: true,
    renderer: renderer
});
```

## 支持的 Markdown 特性

| 特性 | 示例 |
|------|------|
| 标题 | `# H1` `## H2` `### H3` |
| 粗体/斜体 | `**粗体**` `*斜体*` |
| 代码块 | `\`\`\`js\n代码\n\`\`\`` |
| 引用 | `> 引用文本` |
| 列表 | `- 项目` `1. 项目` |
| 表格 | `| 列1 | 列2 |` |
| 链接 | `[文本](?file=路径.md)` |
| 图片 | `![alt](url)` |
| 待办 | `- [ ] 待办` `- [x] 已完成` |

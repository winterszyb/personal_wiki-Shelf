# 前端架构 - 编辑模块

## 核心函数

### 进入编辑模式

```javascript
async function editFile(filePath) {
    const response = await fetch(`/api/wiki/read?file=${encodeURIComponent(filePath)}`);
    const content = await response.text();

    // 切换到编辑器界面
    document.getElementById('content-area').innerHTML = `
        <div class="editor-container">
            <div class="editor-header">
                <h2>编辑: ${filePath.split('/').pop()}</h2>
                <div class="editor-actions">
                    <button class="save" onclick="saveFile('${filePath}')">💾 保存</button>
                    <button class="cancel" onclick="loadFile('${filePath}')">❌ 取消</button>
                    <button class="back" onclick="previewFile('${filePath}')">👁️ 返回浏览</button>
                </div>
            </div>
            <input type="text" value="${filePath}" readonly>
            <div class="editor-toolbar">
                <button onclick="showLinkModal()">🔗 插入链接</button>
                <button onclick="insertText('**粗体**')"><strong>B</strong></button>
                <button onclick="insertText('*斜体*')"><em>I</em></button>
                <button onclick="insertCodeBlock()"><code>{ }</code></button>
            </div>
            <textarea id="editContent">${escapeHtml(content)}</textarea>
        </div>
    `;
}
```

### 保存文件

```javascript
async function saveFile(filePath) {
    const content = document.getElementById('editContent').value;

    const response = await fetch('/api/wiki/update', {
        method: 'PUT',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ file: filePath, content: content })
    });

    const result = await response.json();
    if (result.success) {
        alert('保存成功！');
        loadFile(filePath);  // 切换回预览模式
    }
}
```

### 返回浏览模式

```javascript
function previewFile(filePath) {
    currentFile = null;  // 重置，强制重新加载
    loadFile(filePath);
}
```

## 编辑器工具栏

| 按钮 | 功能 | 实现 |
|------|------|------|
| 💾 保存 | 保存修改 | `saveFile()` |
| ❌ 取消 | 放弃修改 | `loadFile()` |
| 👁️ 返回浏览 | 回到预览 | `previewFile()` |
| 🔗 插入链接 | 图形化选择文件 | `showLinkModal()` |
| **B** | 插入粗体 | `insertText('**粗体**')` |
| *I* | 插入斜体 | `insertText('*斜体*')` |
| { } | 插入代码块 | `insertCodeBlock()` |

## 文本插入函数

```javascript
function insertText(text) {
    const textarea = document.getElementById('editContent');
    const start = textarea.selectionStart;
    const end = textarea.selectionEnd;
    const before = textarea.value.substring(0, start);
    const after = textarea.value.substring(end);

    textarea.value = before + text + after;
    textarea.focus();
    textarea.selectionStart = textarea.selectionEnd = start + text.length;
}
```

## 链接插入模态框

```javascript
async function showLinkModal() {
    document.getElementById('linkModal').classList.add('active');

    // 加载目录树到模态框
    const response = await fetch('/api/wiki/tree');
    const data = await response.json();
    document.getElementById('linkTreeContainer').innerHTML = 
        renderLinkTree(data.tree);
}
```

### 链接插入流程

1. 点击"🔗 插入链接"按钮
2. 弹出模态框，显示目录树（只显示文件）
3. 输入链接显示文本（可选，默认用文件名）
4. 点击目标文件选中
5. 点击"插入"按钮
6. 自动计算相对路径，插入 `[文本](./相对路径.md)`

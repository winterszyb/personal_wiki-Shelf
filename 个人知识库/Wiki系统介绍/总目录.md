# Wiki 系统介绍

## 项目概述

这是一个基于 Markdown 文件系统的个人知识库 Wiki，采用**前后端分离架构**，支持多层级目录管理、Web 端浏览编辑、主题切换等功能。

## 系统架构总览

```
┌─────────────────────────────────────────────────────────────┐
│                        用户浏览器                            │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐ │
│  │  目录树导航  │  │ Markdown预览 │  │   编辑器（编辑模式）  │ │
│  │  - 展开/折叠 │  │  - 代码高亮  │  │   - 源码编辑        │ │
│  │  - 右键操作  │  │  - 内部链接  │  │   - 插入链接工具    │ │
│  └─────────────┘  └─────────────┘  └─────────────────────┘ │
└────────────────────────┬────────────────────────────────────┘
                         │ HTTP API
┌────────────────────────┼────────────────────────────────────┐
│                   Node.js 服务器                             │
│  ┌────────────────────────────────────────────────────────┐ │
│  │  API 路由层                                             │ │
│  │  ├── GET  /api/wiki/tree    → 扫描目录树               │ │
│  │  ├── GET  /api/wiki/read    → 读取文件内容             │ │
│  │  ├── POST /api/wiki/create  → 创建文件/目录            │ │
│  │  ├── PUT  /api/wiki/update  → 更新文件内容             │ │
│  │  ├── DELETE /api/wiki/delete → 删除文件/目录           │ │
│  │  └── PUT  /api/wiki/rename  → 重命名                   │ │
│  └────────────────────────────────────────────────────────┘ │
│  ┌────────────────────────────────────────────────────────┐ │
│  │  文件系统层（Markdown 存储）                             │ │
│  │  个人知识库/                                            │ │
│  │    └── ai-native-builder知识库/                         │ │
│  │          └── clowder-ai架构wiki/                        │ │
│  │                ├── 总目录.md                            │ │
│  │                ├── 01-项目结构.md                       │ │
│  │                └── ...                                  │ │
│  └────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

```bash
# 方式二：命令行
cd D:\文档\wiki
node server.js
```

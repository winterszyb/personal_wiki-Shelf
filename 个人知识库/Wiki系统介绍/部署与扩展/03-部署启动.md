# 部署与扩展 - 部署启动

## 环境要求

| 软件 | 最低版本 |
|------|---------|
| Node.js | 14.x+ |
| 操作系统 | Windows / macOS / Linux |

## 启动方式

### 方式一：双击启动（推荐）

```
双击 "启动服务器.bat"
→ 自动启动 Node.js 服务器
→ 访问 http://localhost:3000
```

### 方式二：命令行

```bash
cd D:\文档\wiki
node server.js
```

```
输出：
Clowder AI Wiki 服务器已启动:
- 本地访问: http://localhost:3000
- Wiki 目录: D:\文档\wiki

按 Ctrl+C 停止服务器
```

## 停止服务

```
双击 "停止服务器.bat"
或 Ctrl+C
```

## 修改端口

编辑 `server.js` 第 7 行：

```javascript
const PORT = 3000;  // 改为你想要的端口
```

## 在其他设备上访问

```bash
# 获取本机 IP
ipconfig  # Windows
ifconfig  # Linux/macOS

# 在其他设备浏览器中访问
http://<你的IP>:3000
```

## 开机自启（Windows）

将 `启动服务器.bat` 添加到 Windows 启动文件夹：

```
Win + R → shell:startup → 放入快捷方式
```

## 生产部署建议

1. **使用 PM2 守护进程**
   ```bash
   npm install -g pm2
   pm2 start server.js --name wiki
   ```

2. **添加 nginx 反向代理**
   ```nginx
   server {
       listen 80;
       server_name wiki.example.com;
       location / {
           proxy_pass http://localhost:3000;
       }
   }
   ```

3. **添加 HTTPS**
   ```bash
   # 使用 Let's Encrypt
   certbot --nginx -d wiki.example.com
   ```

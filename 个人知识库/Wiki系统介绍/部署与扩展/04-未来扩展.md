# 部署与扩展 - 未来扩展

## 功能扩展计划

### 1. 全文搜索

```
实现方案：
  - 后端添加 /api/wiki/search?q=xxx
  - 遍历所有 .md 文件，读取内容匹配
  - 返回匹配结果列表
  - 前端添加搜索框 + 结果展示
```

### 2. 图片上传

```
实现方案：
  - 编辑器工具栏添加"插入图片"按钮
  - 支持拖拽上传或剪贴板粘贴
  - 图片存入 assets/ 目录
  - 自动生成 ![alt](./assets/xxx.png) 链接
```

### 3. 版本历史

```
实现方案：
  - 集成 Git 管理知识库目录
  - 编辑保存时自动 git commit
  - 添加 /api/wiki/history?file=xxx 接口
  - 前端显示修改历史时间线
```

### 4. 导出功能

```
支持格式：
  - PDF：后端使用 puppeteer 渲染
  - HTML：Markdown 转 HTML
  - Word：使用 markdown-to-docx 库
```

### 5. 标签系统

```
实现方案：
  - 在 Frontmatter 中定义标签
  - 添加 /api/wiki/tags 接口
  - 前端添加标签筛选功能
```

### 6. 协同编辑

```
实现方案：
  - WebSocket 实时同步
  - 多用户编辑冲突检测
  - 编辑锁机制
```

### 7. Markdown 实时预览

```
实现方案：
  - 编辑器分屏：左侧编辑 / 右侧预览
  - 实时渲染 Markdown
  - 滚动同步
```

## 技术改进

| 改进 | 说明 |
|------|------|
| TypeScript 迁移 | 类型安全，更好的 IDE 支持 |
| 测试覆盖 | 单元测试 + E2E 测试 |
| 性能优化 | 虚拟滚动（大目录树）、增量渲染 |
| 国际化 | i18n 支持多语言 |

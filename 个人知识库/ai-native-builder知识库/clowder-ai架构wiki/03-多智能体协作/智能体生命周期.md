# 智能体生命周期管理

## AgentService 接口

核心契约：
- `invoke(prompt, options) → AsyncIterable<AgentMessage>`
- 每个提供商通过**派生子进程**运行对应 CLI

## 提供商实现

| 提供商 | CLI |
|---|---|
| Claude | `claude` CLI |
| Gemini | `agy` / `gemini` CLI |
| Codex | `codex` CLI |
| Kimi | Kimi CLI |
| OpenCode | `opencode` CLI |

## 会话延续

决定每个 cat-thread 对是恢复已有会话还是新建。会话链通过 Redis 持久化。

## 注册流程

`index.ts:1270` 中遍历所有 Cat 配置，根据 `clientId` 创建对应服务：

```
'anthropic' → createClaudeAgentServiceForCanary()  (支持 bg_daemon/interactive_pty/print_sdk/api_key 四种载具)
'openai'    → new CodexAgentService()
'google'    → new GeminiAgentService()
```

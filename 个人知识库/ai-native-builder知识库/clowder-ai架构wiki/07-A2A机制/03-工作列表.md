# A2A 工作列表

## 数据结构

```typescript
interface WorklistEntry {
  list: CatId[];                    // 可变工作列表
  originalCount: number;            // 用户选择的目标数
  a2aCount: number;                 // 当前 A2A 深度
  maxDepth: number;                 // 最大深度 (默认 15)
  executedIndex: number;            // 已执行到的位置
  a2aFrom: Map<CatId, CatId>;       // 谁 @ 了谁
  a2aTriggerMessageId: Map<CatId, string>; // 触发消息 ID
  streakPair?: {                    // Ping-pong 追踪
    from: CatId;
    to: CatId;
    count: number;
  };
}
```

## WorklistRegistry

### 注册表
```typescript
// Key: parentInvocationId 或 threadId
const registry = new Map<string, WorklistEntry>();

// 线程级别反向索引
const threadIndex = new Map<string, Set<string>>();
```

### 核心操作

#### 1. registerWorklist
```typescript
registerWorklist(threadId: string, worklist: CatId[], maxDepth: number): WorklistEntry
```
- 调用时机: `routeSerial` 开始时
- 初始化工作列表，设置 `executedIndex=0`

#### 2. pushToWorklist
```typescript
pushToWorklist(threadId: string, callerCatId: CatId, targetCats: CatId[]): void
```
- 调用时机: A2A 回调触发时
- 三重守卫检查:
  1. 调用者授权 (必须是当前执行 Cat)
  2. 深度限制 (`a2aCount >= maxDepth`)
  3. 去重检查 (仅对尾部去重)

#### 3. unregisterWorklist
```typescript
unregisterWorklist(threadId: string, entry: WorklistEntry): void
```
- 调用时机: `routeSerial` 结束时
- 所有权保护: 只能删除自己注册的条目

## 工作列表执行

### 主循环
```typescript
const worklist = [...targetCats];  // 初始化
let index = 0;

while (index < worklist.length) {
  const catId = worklist[index];
  await invokeSingleCat(catId, {
    directMessageFrom: a2aFrom.get(catId),
    streamReplyTo: a2aTriggerMessageId.get(catId)
  });
  
  // A2A 处理，可能扩展 worklist
  const a2aMentions = parseA2AMentions(output);
  for (const mention of a2aMentions) {
    if (shouldRoute(mention)) {
      worklist.push(mention);
      a2aFrom[mention] = catId;
      a2aCount++;
    }
  }
  
  index++;
}
```

### 执行上下文

每个 Cat 收到：
```typescript
{
  directMessageFrom: CatId,       // 谁 @ 了我
  streamReplyTo: messageId,       // 触发消息 ID
  pingPongWarning: boolean,       // 是否处于 ping-pong
  crossThreadReplyHint: {...}     // 跨线程提示
}
```

## 去重策略

### 尾部去重
已执行过的 Cat 可以重新入队：
```
worklist = [A, B, C]
  → A 完成，@B  → B 已在工作列表，不重复添加
  → B 完成，@D  → D 不在工作列表，添加
  → worklist = [A, B, C, D]
```

### 跨路径去重
检查 Cat 是否已在其他路径处理中：
```typescript
if (hasActiveAgent(cat)) {
  return 'skip';  // 跳过该 Cat
}
```

# Clowder AI 项目结构

这是一个 **多智能体协作平台**（cat-cafe），用于在共享持久环境中路由和管理不同 AI 模型之间的任务。

## 整体流程

```
用户消息 → 路由决策 → 上下文组装 → 智能体执行 → A2A检测/回调 → 持久化
```

## 顶层目录

| 目录/文件 | 用途 |
|---|---|
| `packages/` | **Monorepo 核心代码**（5 个包） |
| `desktop/` | Electron 桌面端 |
| `scripts/` | 100+ 运维/开发脚本 |
| `cat-cafe-skills/` | 50+ AI 技能定义 |
| `docs/` | 架构、路线图、决策记录 |
| `assets/` | 品牌词典、Prompt 钩子、系统提示词 |
| `guides/` | YAML 交互指南 |
| `sop-definitions/` | 标准操作流程定义 |

## 5 个核心包 (`packages/`)

| 包 | 技术栈 | 说明 |
|---|---|---|
| **api** | Fastify v4 + Socket.IO + SQLite + Redis | 后端服务，MCP 集成，技能安全，市场 |
| **web** | Next.js 14 + React 18 + Tailwind + Zustand | 前端界面，含代码编辑器、节点图、终端、语音 |
| **shared** | TypeScript + Zod + ioredis | 共享类型、Schema、工具函数 |
| **mcp-server** | MCP SDK v1 | 模型上下文协议服务端 |
| **finance** | 提供者架构 | 金融数据包 |

## 技术栈总览

- **运行时**: Node.js ≥24, pnpm 9.x, TypeScript 5.x
- **后端**: Fastify, Socket.IO, better-sqlite3, Redis, OpenTelemetry
- **前端**: Next.js 14, React 18, Tailwind CSS, CodeMirror, XYFlow, Phaser 3
- **桌面**: Electron 35
- **代码质量**: Biome, ESLint, dependency-cruiser, Vitest

## 关键入口点

- **API 服务**: `packages/api/src/index.ts`
- **Web 前端**: `packages/web/src/app/`
- **MCP 服务**: `packages/mcp-server/src/index.ts`
- **桌面端**: `desktop/main.js`
- **开发启动**: `scripts/start-dev.sh`
